# Dev

1. Clonar el proyecto
2. Copiar el `.env.template` y renombrar a `.env`
3. Asignar valores a las llaves de `.env`
4. Instalar dependencias

```
yarn install
```

5. Levantar el backend de Nest

```
yarn start:dev
```

6. Visitar el sitio [localhost:3000/graphql](http://localhost:3000/graphql)
